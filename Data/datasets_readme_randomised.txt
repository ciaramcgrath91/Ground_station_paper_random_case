README file for Datasets included with Article:
"Data sink selection using consensus leadership: improving target connectivity for a spacecraft constellation"

Description:
-- location of ground stations, targets and spacecraft used as input parameters in the
randomised study described in the above article.

Size:
-- groundstation_locations_randomised.csv = 2kb.
-- target_locations_randomised.csv = 7kb.
-- satellite_coes_randomised.csv = 7kb.

Platform:
-- any

Environment:
-- any

Major component description: Detailed description of the major components of the data set.
-- groundstation_locations_randomised.csv contains the locations of the 77 ground stations used for the ground station selection.
Locations are given in latitude and longitude in degrees.
-- target_locations_randomised.csv contains the locations of the 300 targets used for the study.
Locations are given in latitude and longitude in degrees.
-- satellite_coes_randomised.csv gives the postions of the 100 spacecraft included in the study at the analysis epoch.
Positions are given in Keplerian orbital elements.
